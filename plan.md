## UI-полировка (сделано в процессе)

- **Hero CTA кнопка:** `@keyframes cta-glow` — пульсирующий синий ореол, при hover анимация стоп

- **Кнопка Авито:** `@keyframes avito-pulse` — расходящийся ореол на границе, +scale при hover
- Все анимации уважают `prefers-reduced-motion`

## Технические заметки

- Файл смешанные окончания строк (LF/CRLF); Edit-инструмент не матчит — все правки через PowerShell `[System.IO.File]::ReadAllText/WriteAllText` UTF-8; многострочные замены — regex с флагом Singleline
- `grid-template-columns: 360px minmax(0, 1fr)` — `minmax(0,...)` обязателен, иначе колонка расширяется до min-content (~1882px)
- JS карусели: `flexBasis` (не `width`), init через `requestAnimationFrame`
- SVG-путь телефона (`phone` Feather Icons): правильный путь с `stroke-linecap="round" stroke-linejoin="round"`
- Фото `img/specialist.jpg` — 6.5 МБ, перед публикацией сжать до ~150 КБ WebP
