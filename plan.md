## Что реализовано (факт)

### ✅ Шаг 1 — Скелет: Header + Hero + Ticker

**Файл:** `index.html` (единый файл, без внешних CSS/JS)

**Сделано:**
- CSS-переменные: вся дизайн-система из `brief.md` (цвета, шрифты, отступы, тени)
- Google Fonts: Manrope + Inter с кириллицей
- **Header:** логотип ТОНУС крупный (1.75rem, bold, синий) + разделитель + подпись; nav с underline-hover; sticky — стекло при скролле (backdrop-filter)
- **Мобильный header:** burger с анимацией крестика, выпадающее меню
- **Hero:** светлый градиентный фон (голубой → белый), двухколоночная раскладка
  - Заголовок с градиентным текстом: «Снова двигаться без боли — это реально»
  - Badge-бейджик с пульсирующей точкой
  - CTA + кнопки мессенджеров
  - Доверие-якорь (★★★★★, годы, пациенты)
  - Параллакс-декорации (3 пятна, разная скорость)
  - Плавающие бейджи «Лечение с первого приёма» и «10+ лет практики»
- **Фото специалиста** (Маша) — реальное фото в blob-рамке (460×560px), object-fit: cover top
- **Blob:** морфинг формы 18s, без float-анимации (убавленная динамика), кольца статичные
- **Ticker:** CSS-only бегущая строка, синий фон
- **Анимации загрузки:** hero-контент появляется снизу вверх с stagger-задержками
- **Scroll Reveal:** IntersectionObserver для всех `.reveal`
- **Параллакс:** 3 декоративных пятна движутся с разной скоростью
- **Плавный скролл** по якорям из меню
- **prefers-reduced-motion** — все анимации отключаются

### ✅ FAQ-секция (добавлена досрочно по запросу)

**6 вопросов-ответов в аккордеоне:**
1. Что такое остеопатия? (полный текст с принципами)
2. Что такое биодинамика? (полный текст)
3. Чем вы отличаетесь от массажиста?
4. Нужно ли направление от врача или МРТ?
5. Как проходит первый приём?
6. Сколько стоит первый приём?

**Механика:** один открытый за раз, плавное раскрытие max-height, rotate-стрелка, aria-expanded
