## Детали каждого шага

### ✅ Шаг 1 — Скелет: Header + Hero + Ticker

**Что сделано:**
- Единый `index.html`: весь CSS в `<style>`, весь JS в `<script>`
- CSS-переменные: цвета, шрифты, отступы из `brief.md`
- Google Fonts: Manrope + Inter
- Header: прозрачный → матово-белый при скролле (backdrop-filter)
- Мобильный burger-меню с анимацией
- Hero: тёмно-синий градиент, двухколоночный лейаут, blob-плейсхолдер для фото
- Кнопки: главная CTA + три мессенджера + доверие-якорь
- Ticker: CSS-only бегущая строка на синем фоне
- Scroll Reveal: IntersectionObserver для всех `.reveal`
- prefers-reduced-motion поддержка

**Результат в браузере:** открывается красивая hero-секция сразу.
