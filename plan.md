### ✅ Шаг 1 — Скелет: Header + Hero + Ticker

**Файл:** `index.html` (единый файл, без внешних CSS/JS)

**Сделано:**

- **Мобильный header:** burger с анимацией крестика, выпадающее меню
- **Hero:** светлый градиентный фон (голубой → белый), двухколоночная раскладка

  - Badge-бейджик с пульсирующей точкой

  - Доверие-якорь (★★★★★, годы, клиенты)
  - Параллакс-декорации (3 пятна, разная скорость)
  - Плавающие бейджи «Работа с первого сеанса» и «10+ лет практики»
- **Фото специалиста** (Мария Федина) — реальное фото в blob-рамке (460×560px), object-fit: cover top
- **Blob:** морфинг формы 18s, без float-анимации (убавленная динамика), кольца статичные

- **Плавный скролл** по якорям из меню
- **prefers-reduced-motion** — все анимации отключаются

### ✅ Hero — 3D-эффект фото (доработка)

**Сделано:**
- Blob: утолщённая рамка-проём + три кольца внешнего свечения
- `::after` псевдоэлемент: inset-тени по краям + блик сверху-слева → ощущение заглублённого проёма в стене
- JS mouse-parallax (десктоп): blob наклоняется ±12° за курсором по всему hero, фото смещается в обратную сторону (16px) — классический "looking through a window"
- Плавность через requestAnimationFrame + lerp (k=0.07)
- На touch-устройствах и при prefers-reduced-motion — эффект отключён
